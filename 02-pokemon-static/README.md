# Pokemon Static

Sitio web estático con los primeros 151 Pokémon, construido con Astro y Tailwind CSS.

## Características

- Lista de Pokémon con imágenes y sprites oficiales
- Página individual para cada Pokémon con su sonido característico
- Generación de sitio estático (SSG) para máxima velocidad
- View Transitions para navegación fluida
- Diseño responsivo con Tailwind CSS

## Comandos

```bash
# Instalar dependencias
pnpm install

# Servidor de desarrollo
pnpm dev

# Construir sitio para producción
pnpm build

# Vista previa del build
pnpm preview
```

## Tecnologías

- **Astro 5** - Framework para sitios estáticos
- **Tailwind CSS 4** - Estilos
- **TypeScript** - Tipado
- **PokeAPI** - Datos de Pokémon
